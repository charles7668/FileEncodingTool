# FileEncodingTool

Encoding tool for file

# Usage

```shell
Usage:
  FileEncodingTool [options]

Options:
  -d, --directory <directory>           target directory
  -t, --target <target>                 Convert target encoding; if not specified, only detect files.
  -p, --patterns <patterns> (REQUIRED)  To detect files with a search patterns.
  --sub-dir                             Include subdirectories, default is true.
  --version                             Show version information
  -?, -h, --help                        Show help and usage information
```
